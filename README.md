# Interview Ready 🚀

Interview Ready is a web-based interview preparation platform designed to help candidates practice and improve their interview skills. The project includes candidate registration/login, mock tests, resume upload, AI voice interview support, and result tracking.

## Features

✅ Candidate Registration & Login  
✅ Candidate Profile Management  
✅ Mock Test System  
✅ Test Submission & Result Evaluation  
✅ Resume Upload Feature  
✅ AI Voice Interview Module  
✅ Candidate List Management  
✅ Secure Logout System  
✅ Responsive UI Design  

## Project Modules

- **Authentication System**
  - Candidate Registration
  - Candidate Login
  - Logout

- **Profile Management**
  - View/Edit Candidate Profile

- **Mock Interview Tests**
  - Practice Mock Tests
  - Submit Answers
  - View Results

- **Resume Management**
  - Upload Resume Files

- **AI Interview**
  - Voice-based AI Interview Simulation
